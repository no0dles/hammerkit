# CLI

## Installation / Upgrade

The hammerkit cli can be easily installed and upgraded by npm/yarn.

{% tabs %}
{% tab title="npm" %}
```bash
npm i -g hammerkit
```
{% endtab %}

{% tab title="yarn" %}
```bash
yarn add -g hammerkit
```
{% endtab %}
{% endtabs %}
